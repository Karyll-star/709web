# 709寝室宇宙 - 响应式网站

一个充满趣味和创意的寝室主题网站，采用现代化的响应式设计，能够自动适配各种设备和屏幕尺寸。

## 🌟 主要特性

### 🎨 响应式设计
- **多设备适配**: 完美支持手机、平板、电脑等不同设备
- **智能布局**: 根据屏幕尺寸自动调整网格列数和元素大小
- **触摸友好**: 移动端优化的触摸交互体验

### 📱 移动端优化
- **汉堡菜单**: 移动端折叠式导航菜单
- **触摸手势**: 支持拖拽、点击等触摸操作
- **自适应字体**: 根据屏幕大小调整文字大小

### 🖥️ 桌面端体验
- **左侧外链栏**: 大屏幕显示便捷的外链入口
- **悬停效果**: 丰富的鼠标悬停动画效果
- **宽屏布局**: 充分利用大屏幕空间

## 🚀 响应式断点

| 断点 | 屏幕宽度 | 网格列数 | 特性 |
|------|----------|----------|------|
| 默认 | < 640px | 2列 | 移动端优化 |
| sm | ≥ 640px | 3列 | 小屏平板 |
| md | ≥ 768px | 4列 | 平板横屏 |
| lg | ≥ 1024px | 5列 | 小屏电脑 |
| xl | ≥ 1280px | 6列 | 大屏电脑 |

## 🎨 设计原则

### 响应式设计
1. **移动优先**: 从移动端开始设计，逐步增强
2. **内容优先**: 确保内容在所有设备上都清晰可读
3. **性能优化**: 根据设备能力调整功能复杂度
4. **一致性**: 保持跨设备的视觉和交互一致性

### 用户体验
1. **直观导航**: 清晰的导航结构和反馈
2. **触摸友好**: 适合触摸操作的按钮和交互元素
3. **快速响应**: 流畅的动画和过渡效果
4. **无障碍访问**: 支持键盘导航和屏幕阅读器

## 🔧 自定义配置

### 响应式断点调整
```css
/* 在 src/style.css 中修改断点 */
@media (min-width: 640px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 1024px) { /* lg */ }
@media (min-width: 1280px) { /* xl */ }
```

### 网格列数配置
```javascript
// 在 src/script.js 中调整网格列数
function getComputedCols() {
    const w = window.innerWidth;
    if (w >= 1280) return 6;      // xl: 6列
    if (w >= 1024) return 5;      // lg: 5列
    if (w >= 768) return 4;       // md: 4列
    if (w >= 640) return 3;       // sm: 3列
    return 2;                      // 默认: 2列
}
```
